# Dataset citation

When using or discussing results based on this corpus, please cite:

> Morgan, K., Pickering, C., Goodwin, M., Wu, H., Ghanta, M., Gupta, A., Goldenholz, D., & Westover, M. B. (2026). *The Neurotech EEG Dataset* (version 1.0). Brain Data Science Platform. https://doi.org/10.60508/v99k-ek82

- BDSP access: credentialed users + DUA  
- Official BIDS / de-identification code (dataset authors, **not this ML repo**): https://github.com/bdsp-core/Neurotech-EEG-Wrangling  
- Dataset limitations (from the BDSP page): single provider/hardware; annotations are **clinical workflow annotations, not multi-expert research labels**; filter header-only stub EDFs with `n_records > 0`.
